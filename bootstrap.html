Bootstrap Menu

This is another usage of the Navbar.js initialized to the menu itself, just like in the Left Menu demo, but this time we're using the original Bootstrap Navbar wrapper and markup. This is like a hybrid usage of the Navbar.js with the least possible CSS redundancy.

Demo Highlights

  • This demo features a special Navbar.js CSS layout, it doesn't use its wrapper or markup, as we'll see in a minute, but the full Bootstrap 4 / 5 original Navbar markup and the initialization target is the <ul class="navbar-nav"> element.
  • The idea of this demo is to showcase how to integrate Navbar.js with Bootstrap while keeping the original design concept of Bootstrap. Basically the CSS applies to the submenu and its animations, submenu items and menu icons.
  • We've enabled the toggleSiblings option via the specific attribute data-toggle-siblings="true".
  • Also for this demo we've set the responsive breakpoint to 768px, via navbar-expand-md class on the Bootstrap side, which is also the default for Navbar.js, but that's up to you to adjust for your need.
  • For this demo we've developed dedicated SCSS mixins where most of the variables are ignored.

Bootstrap Navbar Wrapper

<nav class="navbar navbar-dark bg-dark">
  <a class="navbar-brand" href="#">Navbar.js</a>
  <button class="navbar-toggler" type="button" data-toggle="collapse" data-target="#navbar-wrapper">
    <span class="navbar-toggler-icon"></span>
  </button>
      
  <div id="navbar-wrapper" class="navbar-collapse collapse">
    <ul class="navbar-nav me-auto" data-function="navbar" data-breakpoint="768" data-toggle-siblings="true">
      <li class="nav-item"><a class="nav-link" href="#">Root Menu Item</a></li>
      <li class="nav-item active">
        <a class="nav-link" href="#">
          Parent Menu Item
          <i class="parent-icon"></i>
          <i class="parent-toggle"></i>
        </a>
        <ul class="subnav">
          <li class="nav-item active"><a class="nav-link" href="#">Submenu Item 1</a></li>
          <li class="nav-item"><a class="nav-link" href="#">Submenu Item 2</a></li>
        </ul>
      </li>
    </ul>
    <span class="navbar-text">
      <a href="#">Some link</a> and other text.
    </span>
  </div>
</nav>

As you can see we're using the original plugin markup and functionality with the collapsible element, as well as the .navbar-nav class for the menu itself, the .nav-item and .nav-link classes for menu items and .navbar-text class for other contents.

Bootstrap Menu Mixins

// bootstrap.scss - mixins
@import 'variables','mixins','mixins-bs';

@include navbar-style-bs;
@include nav-layout-bs;
@include main-menu-bs;
  • for this build we're using the dedicated mixins set from the mixins-bs.scss file;
  • the navbar-style-bs applies menu items text color and submenu background color for both .navbar-dark (this very example) and .navbar-light;
  • nav-layout-bs applies the submenu layout;
  • main-menu-bs sets the position of submenu and animations according to the layout.

Used Variables

// bootstrap.scss - custom variable input values
$primary_color: #007bff;
$item_line_height: 24px;

$subnav_item_hover_color: #262626;
$subnav_item_hover_background: #ddd;

$subnav_item_active_color: #fff;
$subnav_item_active_background: $primary_color;

$navbar_padding: 0.5rem 1rem;
$subnav_padding: 0.5rem;
  • $primary_color we've set the accent color as the original Bootstrap $brand-primary;
  • $item_line_height to match Bootstrap original typography;
  • next we've set $subnav_item_hover_color and $subnav_item_hover_background;
  • then the $subnav_item_active_color and $subnav_item_active_background;
  • $navbar_padding sets the navbar padding exactly as for the original navbar layout; the reason for using this variable here is because it's important to determine an offset for the level 1 submenu;
  • lastly, we set a $subnav_padding for the FIRST level submenu, just because the looks.
