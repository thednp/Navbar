Bootstrap Extended Navbar

Unlike the other Bootstrap demo this one showcases how to easily integrate Navbar.js with its full features and markup with Bootstrap. Only a couple of variables are required with your special flavor and you're ready to go.

Demo Highlights

  • The build showcases how to recreate the other demo just by using the default Navbar.js markup and adjusting some variables. It might strike you that you can simply replace the original Bootstrap 4/5 Navbar JavaScript plugin and its CSS sources completely, since Navbar.js doesn't require Dropdown and Collapse components (and their dependencies) in order to work.
  • The idea of this demo was to showcase that Navbar.js can create a more efficient workflow, thanks to the more simple and straightforward markup, a light footprint and powerful SCSS mixins that enable real flexible design.
  • We've used the same default responsive breakpoint of 768px, and this is another cool thing, you set one value in your SCSS variables and your JavaScript options exactly for your need, and without ALL the other breakpoints that Bootstrap Navbar comes with.

Bootstrap Extended SCSS

In terms of SCSS, this demo is very much the same with the default layout, we only removed a series of style rules with a couple of specific SCSS variable values, just because we can.

// bootstrap-extended.scss - mixins
@import 'variables','mixins';

@include navbar-mobile-layout;
@include navbar-desktop-layout;
@include navbar-style;
@include nav-layout;
@include nav-style;
@include subnav-style;
@include main-menu;

As you can see, so far bootstrap-extended.scss is identical with navbar.scss, we can move along.

// bootstrap-extended.scss - custom variable input values
$primary_color: #007bff;

$navbar_background: #343a40;
$subnav_background: #e4ebf3;

$navbar_padding: 0.5rem 0;
$subnav_padding: 0.5rem;

$root_item_hover_background: transparent;
$root_item_hover_color: auto;

$root_item_active_background: transparent;

$subnav_item_hover_color: rgba(0,0,0,.75);
$subnav_item_hover_background: rgba(0,0,0,.125);

$subnav_item_active_color: $primary_color;
$subnav_item_active_background: transparent;

$brand_link_background: transparent;
$brand_link_background_hover: transparent;
  • similar to the other Bootstrap demo we've set a $navbar_padding, just for the same reason;
  • the rest of it is for the removal of background colors, for all active, hover and default state.
